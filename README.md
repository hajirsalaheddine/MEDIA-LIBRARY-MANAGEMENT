# Media Library Management 

A Java application demonstrating OOP concepts through a simple media library.

---

## Features

Manage Books (Livre) and DVDs (DVD)
Inheritance: Media as parent class
Polymorphism: Overridden afficherInfo()
Store all media in an ArrayList<Media>
Display detailed information for each item

---

### Structure
src/
├── Media.java
├── Livre.java
├── DVD.java
└── MediaLibrary.java


---

#### Run

Open in Eclipse → run MediaLibrary.java as Java Application

Sample Output:

Book Nº101: The Little Prince written by Saint-Exupéry
DVD Nº202: Inception - Duration: 148 min
Book Nº303: Java for Dummies written by B. Burd
